# User CRUD App

This is a simple **Node.js + MySQL** application to manage users.  
It allows you to:

- View all users
- Add a new user
- Edit a user's username (requires password)
- Delete a user (requires email & password)

## Technologies Used

- Node.js
- Express.js
- MySQL
- EJS for templating
- Faker.js for dummy user data
